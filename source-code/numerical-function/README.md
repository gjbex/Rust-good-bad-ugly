# Numerical function

This application illustrates how to

1. define a simple function that takes several double precision floating point
   arguments and returns a single double precision floating point value;
1. loop over integer values;
1. how to convert integers to double precision floating point values in
   arithmetic expressions;
1. how to format double precision floating point values when printing.


## What is it?

1. `src/main.rs`: main source file for the application.
1. `Cargo.toml`: configuration file for the Rust package manager, including the
   dependency on `clap`.
1. `Cargo.lock`: lock file for the Rust package manager, automatically
   generated when building the application, which specifies the exact versions
   of all dependencies used in the project.


## How to use?

Run the application with the following command to see the help message:

```bash
cargo run --  --help
```
