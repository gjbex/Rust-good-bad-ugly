# Math

This application illustrates arithmetic operators and mathematical functions
for integer and floating-point values.

It shows how to:

* use `+`, `-`, `*`, `/`, and `%` with integers;
* compare ordinary integer division and remainder with Euclidean division and
  remainder for negative values;
* use `+`, `-`, `*`, `/`, and `%` with floating-point values;
* call mathematical methods on `f64`, such as `sin`, `cos`, `sqrt`, `powi`,
  `powf`, `exp`, `ln`, and `log10`;
* use rounding and absolute-value methods such as `abs`, `floor`, `ceil`,
  `round`, and `trunc`.


## What is it?

1. `src/main.rs`: main source file for the application.  It prints examples of
   integer arithmetic, floating-point arithmetic, and mathematical functions.
1. `Cargo.toml`: configuration file for the Rust package manager.
1. `Cargo.lock`: lock file for the Rust package manager, automatically
   generated when building the application.


## How to use?

Run the application:

```bash
cargo run
```
