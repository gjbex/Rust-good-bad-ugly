# Basic types

This application illustrates Rust's basic types, integers, floating point
numbers, booleans, characters and address types.  It shows also how to format
output for floating point numbers and how to use mathematical constants for
`f32` and `f64`.


## What is it?

1. `src/main.rs` contains the main function of the application.  It defines
   some variables of basic types and prints them to the console.
1. `Cargo.toml` is the manifest file for Rust's package manager, Cargo.  It
   specifies the name, version, and dependencies of the project.
1. `Cargo.lock` is automatically generated by Cargo to keep track of the exact
   versions of dependencies used in the project.  It ensures that the same
   versions are used across different builds and environments.


## How to use?

To run the application, simply run:

```bash
cargo run
```
