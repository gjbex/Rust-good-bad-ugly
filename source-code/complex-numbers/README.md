# Complex numbers

This application illustrates how to work with complex numbers in Rust.  Compared
to `../basic-types`, it shows how to:

* use the external `num-complex` crate;
* import a type from a crate with `use`;
* create complex numbers with `Complex64`;
* add and multiply complex numbers;
* access the real and imaginary parts of a complex number;
* compute the norm of a complex number.

Complex numbers are not part of Rust's standard library, so scientific
applications typically use a crate such as `num-complex`.


## What is it?

1. `src/main.rs` contains the main function of the application.  It creates two
   complex numbers, performs arithmetic on them, and prints their components
   and norm.
1. `Cargo.toml` is the manifest file for Rust's package manager, Cargo.  It
   specifies the dependency on `num-complex`.
1. `Cargo.lock` is automatically generated by Cargo to keep track of the exact
   versions of dependencies used in the project.  It ensures that the same
   versions are used across different builds and environments.


## How to use?

To run the application, simply run:

```bash
cargo run
```
