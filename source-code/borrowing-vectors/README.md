# Borrowing vectors

This example introduces borrowing for vector-like data.  Compared to
`../mutable-borrowing`, it shows how to:

* store values in a `Vec<f64>`;
* pass read-only borrowed data as a slice, `&[f64]`;
* pass mutable borrowed data as a mutable slice, `&mut [f64]`;
* iterate over a slice with `iter` and `iter_mut`;
* use iterator methods such as `sum`;
* see why Rust does not allow a mutable borrow while a shared borrow is still
  in use.


## What is it?

1. `src/main.rs` contains the main function of the application.  It computes
   the mean of a vector, normalizes the data in place, and includes commented
   code that demonstrates a borrow-checker error.
1. `Cargo.toml` is the manifest file for Rust's package manager, Cargo.  It
   specifies the name, version, and dependencies of the project.
1. `Cargo.lock` is automatically generated by Cargo to keep track of the exact
   versions of dependencies used in the project.  It ensures that the same
   versions are used across different builds and environments.


## How to use?

To run the application, simply run:

```bash
cargo run
```
